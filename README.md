## Telepítés

**Windows:** Futtasd a Setup .exe-t, majd indítsd el a GYROCUELaser programot.

**macOS:** Nyisd meg a DMG fájlt, húzd az alkalmazást az Applications mappába.
Első indításnál engedélyezd a szükséges jogosultságokat:
System Settings → Privacy & Security → Accessibility és Input Monitoring

---

## Használat

- **F18** (lenyomva tartva) – lézer megjelenítése
- A panel ablakban állítható a szín, méret és kezdőpozíció

---

## Build

**Windows:**
```
build_all.bat
```

**macOS:**
```
./build_mac.sh
```
